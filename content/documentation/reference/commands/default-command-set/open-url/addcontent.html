Open a page by using an absolute URL

Open url 'http://www.wetator.org/commands/openurl.html'.

Command Parameter Optional Parameter
open-url http://www.wetator.org/commands/openurl.html
assert-title WETATOR / Open Url

Open a local file

{{% image src="docu-cmd-openurl-file.png" %}}

Open file 'c:\wetator\commands\openurl.html'.

Command Parameter Optional Parameter
open-url file://c:/wetator/commands/openurl.html
assert-title WETATOR / Open Url
