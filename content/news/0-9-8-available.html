Wetator version 0.9.8 is available.

This release no longer supports Firefox 3 simulation and brings an experimental Firefox 10 (ESR) support.
Again we use the latest HtmlUnit snapshot which brings a newer Javascript implementation together with many bug fixes.
Please have a look at the release notes.

Happy testing...
