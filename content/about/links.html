Used libraries

Wetator uses many open source libraries. Writing a tool like this is impossible without the uncountable effort of all the people who have contributed to one of these:
  • HtmlUnit
  • CSS Parser
  • Apache commons (lang, io, logging, codec)
  • Apache PDFBox
  • Apache POI
  • dk.brics.automaton
  • Txtmark
  • The Legion of the Bouncy Castle
