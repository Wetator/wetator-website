Wetator Release 1.2 is available.

Wetator 1.2 is available. Please note that the list of supported browsers has changed (FF38 was released some days ago). Starting with this version we only support IE 8, IE11, Firefox 31(ESR), Firefox 38(ESR), and Chrome as browsers.
This release uses the HtmlUnit 2.17 release.
Please have a careful look at the release notes.

Enjoy smart testing...
