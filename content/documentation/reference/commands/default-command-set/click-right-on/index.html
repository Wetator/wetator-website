{{% cmd-register-cs-hint %}}

Right click with the mouse on the specified element.

Parameters

wpath
The WPath identifying the control to right click on {{% link-wpath-docu %}}.
It is possible to use the '*' and the '?' character als wildcards; '*' matches zero or many characters; '?' matches one single character.
It is also possible to define a {{% wpath-separator %}} separated list of words (with '*' and '?' if needed). Then this command clicks on the first double clickable element after all these words.

Examples

Right click an entry field

Right click on an entry field labled 'Name'.

Command Parameter Optional Parameter
click-right-on Name
