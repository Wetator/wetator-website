Wetator Release 1.17 is available.

Wetator 1.17 is available.
This release uses the HtmlUnit 2.56.0 release. The updated HtmlUnit brings improved javascript support based on the latest Rhino code and some fixes. Also the usual updates to be in sync with the latest versions or Chrome and Firefox.
Firefox78 is now replaced by FirefoxESR. Please have a lock at your configuration and adjust if required.
Please have a careful look at the release notes.

Additionally log4j is updated to version 2.16.0 to address the security issue.

Enjoy smart testing...
