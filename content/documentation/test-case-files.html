Wetator test cases are stored in test case files as a table where every line contains a command.

There a three different file formats supported for writing down your test cases out of the box:

  • Wetator XML (*.wet)
  • WikiText (*.wett)
  • Excel (*.xls, xlsx)

The details of these formats, as well as the main benefits and problems with them, are explained on the subsequent pages. Which format to use depends on your context.

Please find sample files using the different formats inside the 'templates' folder of the distributed zip file.
