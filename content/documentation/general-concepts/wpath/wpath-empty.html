An empty WPath simply means:
Please use the first matching control on the page. This might be handy for pages similar to the well known search engines.

{{% image src="docu-wpath-empty.png" %}}
Command Parameter Optional Parameter
set wetator

Furthermore you can use WPaths without a last selector like this:

Command Parameter Optional Parameter
click-on Headline >

This will click on the first clickable element after 'Headline'.
