For getting started please follow our {{% pagelink "installation" "installation guide" %}} and have a look at {{% pagelink "write-your-first-wetator-test" "how to write your first Wetator test" %}}.

Beyond running Wetator from {{% pagelink "command-line-options" "command line" %}} you also have the possiblity to run Wetator via {{% pagelink "ant-task" "ant task" %}}.

Furthermore Wetator comes with

  • the Wetator Test Editor (WTE) - an Eclipse plugin for convenient editing of test files
  • and a Jenkins plugin to make Wetator part of your Continuous Integration.

Visit the {{% pagelink "download" "download section" %}} to get them.
